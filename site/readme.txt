
Скрипт для вывода информации по курсу доллара с сервера www.cbr.ru
в виде графического информера
Язык реализации: PHP4
Copyright (c) 2001, Mikhail Turenko, www.turenko.net, <mikhail@turenko.net>
Бесплатно для любого использования.

Файлы:

usd.php - выводит PNG картинку 88x41 вида
	dd/mm/yyyy
	USD/RUR
	29.30 (+0.02)
	пример использования:
	<IMG src="usd.php" width="88" height="41" border="0"
	alt="Курс доллара ЦБ РФ" />
	рабочий пример: http://img.intermac.ru/usd

usd2.php - выводит PNG картинку переменного размера вида
	dd/mm/yyyy: 1USD=29.30RUR (+0.02)
	пример использования:
	<IMG src="usd.php" border="0" alt="Курс доллара ЦБ РФ" />
	рабочий пример: http://img.intermac.ru/usd2
